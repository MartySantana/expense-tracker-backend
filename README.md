# Expense Tracker Backend

API REST développée avec Spring Boot permettant :

- Ajouter une dépense
- Lister les dépenses
- Filtrer par catégorie
- Filtrer par période
- Calculer le total des dépenses
- Calculer le total par catégorie

## Technologies

- Java 17
- Spring Boot
- Maven

## Lancement

```bash
mvn spring-boot:run
```
